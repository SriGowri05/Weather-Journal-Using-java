# Weather Journal

## Project Description
Weather Journal is a Java Swing application that allows users to record daily weather information, view saved entries, and calculate average temperature and humidity.

## Features
- Add weather entries
- View saved entries
- Calculate average temperature
- Calculate average humidity
- User-friendly GUI

## Technologies Used
- Java
- Swing
- AWT
- ArrayList

## How to Run

1. Install Java JDK.
2. Open the project in VS Code, Eclipse, or NetBeans.
3. Compile and run `WeatherJournal.java`.
